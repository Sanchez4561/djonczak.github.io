About level

This is level that was prepared during Level Design classes. The level was to be made using methods from this video and with using assets from Assets Store. The player moves around the closed cemetery at night and his goal is to find all items and path, that will let him got to the main goal, which is indicated by a large statue highlighted in red. Using level characteristic points, lights, terrain and other visual elements. In addition, we had to let player save the game and add the possibility of changing game language to Polish, German or English. For languages we had to use Scriptable Objects and for game saves we had to save the file in json format, in my case it stored the player position and picked items.
